# About 
The project is simple but complex, an idea so easy on paper and difficult to execute... The project was fragmented into 2 stages, Initial Compiler and Final Compiler. The initial compiler is written in C, the final one will be written in BFlex. This will allow our project to be tested by ourselves at creation time.
